Installation

Given below are the steps you need to follow, to install the Dleohr Template - React on your system:

Step 1: Open the terminal in your root directory & to create the project run the following command:

npx create-react-app Project_name

Step 2: By running the following command, you will be able to get all the dependencies in your node_modules folder:

npm install

Step 3: To run the project, you need to run following command in the project directory. It will compile the jsx files & all the other project files. If you are making any changes in any of the .jsx file then you need to save the file, it will automatically consider this command and run the entire code.

npm run dev

Step 4: To install any plugins in your application, you have to run the following command

npm install plugin_name --save

Building for Production: If you want to run the project and make the build in the production mode then run the following command in the root directory, otherwise the project will continue to run in the development mode.

npm run build

Create New Page

How to add New Page:
  • Create New View with index.js extenstion.
  • Every page of code must be written inside the index.jsx. In this may occure the conflict, so we have to create the folder of new page, inside the folder create the index.jsx file
  • Set Path in routes/foldername/index.jsx file to serve the page on the browser.
Steps to add a new page:

Step 1: Create a file with the extension index.jsx under the src -> routes -> foldername directory. Let's create a testPage for an example with filename testPage/index.jsx and placing the below code in that file.


import React, { Component } from 'react';
export default class newPage extends Component {
  constructor(props) {
    super(props);
    this.state = {}
  }	
  render() {	
   return (  );
  }
}

Step 2: After creating file,you need to declare its route where it can be served on the browser, suppose you want created page to be serve on the route http://localhost:3000/newPage . To access this page define its routes in the routeservice -> index.jsx file.


import React from 'react';
import { Redirect, Route, Switch } from 'react-router-dom';
import NewPage  from './newPage';
const NewRoute = () => (
   <Switch>
      <Route path="/new-page" component={NewPage} />
   </Switch>
);
export default NewRoute;

After completing these above steps you need to run the command npm run dev or npm start command in the project directory.It will serve your app on the localhost.
