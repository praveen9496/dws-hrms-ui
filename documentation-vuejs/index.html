Installation Guide

  • For developing a vuejs project,node and npm mandatory.To install https://nodejs.org/en/download/
  • Step:1 Open the terminal in your root directory,By running the following command you will get a webpack vue-cli: npm install -g vue-cli
  • Step 2: Open the terminal in your root directory,By running the following command you will get a full webpack vuejs project folder: vue init webpack my-project
  • Step 3: Open the terminal,By running the following command you will goto project installed directory: cd my-project(my-project is which we installed project name.)
  • Step 4:By running the following command, you will be able to get all the dependencies in your node_modules folder: npm install
  • Step 5:To run the project, you need to run following command in the project directory.npm run dev
  • Step 6: If you want to make the build in the production mode then run the following command in the root directory. npm run build
